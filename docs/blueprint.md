# **App Name**: SnackDash

## Core Features:

- Menu Browsing: Browse the available snack menu with detailed descriptions and images.
- Shopping Cart: Add and manage items in a shopping cart.
- Secure Payment: Process payments securely via Stripe.
- Order Tracking: Track the real-time status and location of your order.
- Suggested Snack Combinations: AI tool suggests snack combinations for users based on what they have already ordered, maximizing the fun for users by combining snack options.

## Style Guidelines:

- Primary color: Vibrant orange (#FF9500) to evoke appetite and energy.
- Background color: Soft yellow (#FFF4D4) to provide a bright and welcoming backdrop.
- Accent color: Black (#000000) to create strong contrast and highlight important elements.
- Body and headline font: 'Poppins' (sans-serif) for a modern and easily readable style.
- Use clear, food-related icons for menu items and actions.
- Clean and intuitive layout to enhance user experience.
- Subtle animations for transitions and order confirmations.
